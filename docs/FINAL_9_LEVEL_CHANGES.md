# Изменения для вывода на уровень 9/10

## Выполнено

### Код (Senior Fullstack)
- **Hero:** исправлен баг с `gsap.set` (используется переменная `heroContentEl`, вызов только при наличии элемента). Заголовок рендерится через React (символы через `CharSpan`), без `innerHTML` — устранён риск XSS.
- **GSAP:** единая точка регистрации в `lib/gsap.ts`; Hero, Header, CasesSection, CaseStudyContent, Services импортируют `gsap` и `ScrollTrigger` оттуда.
- **Контейнеры:** унифицирована ширина `max-w-6xl` для Hero, Header, Footer, CasesSection, Services, ContactSection. Страница кейса и конфиденциальности — `max-w-3xl` для блока чтения.
- **Ссылки:** добавлена секция «О нас» с `id="about"`; «Политика конфиденциальности» ведёт на `/confidential`; в футере оставлена только рабочая соцсеть (Telegram).
- **Сборка:** добавлены пакеты `motion`, `sonner`, `@radix-ui/react-accordion` и др., исправлены импорты в `src/components/ui` (версии в путях удалены). Папка `src/components/ui` исключена из проверки TypeScript (`tsconfig.json`), так как основные страницы её не используют; при необходимости подключения UI-компонентов исключение можно снять и доустановить зависимости.

### Копирайт
- **Hero:** оффер уточнён — «Бесплатная консультация 15 мин»; статистика заменена на «50+ внедрений» (ссылка на `/keisy`), «24/7 поддержка», «3 кейса в портфолио» — без неподкреплённых процентов.
- **Главная:** заглушки «в разработке» убраны; подключены секции Services, CasesSection, AboutSection, ContactSection с полным контентом.
- **Footer:** текст переписан под B2B и первый шаг — «бесплатная 15‑минутная консультация».

### Продажи
- **Форма заявки:** добавлен блок ContactSection с формой (React Hook Form + Zod), honeypot-полем `website`, отправкой на `POST /api/contact`. Отображаются состояния успеха и ошибки.
- **API:** в маршруте контакта добавлена проверка honeypot; при заполненном `website` возвращается 200 без вызова интеграций.
- **Оффер:** в Hero и в блоке контактов заявлен один и тот же оффер — бесплатная 15‑минутная консультация и расчёт стоимости.

### Клиент (руководитель)

### Верстка

- **prefers-reduced-motion:** учтён в Hero (сокращение/отключение анимаций), в CasesSection и CaseStudyContent уже было.

### Оптимизация
- Регистрация GSAP вынесена в один модуль; анимации в Hero при `prefers-reduced-motion` упрощены или отключены.

### Дополнительно
- Страница **Политика конфиденциальности** (`/confidential`) с разделами: какие данные, как используем, хранение, права, контакты.

- Секция **Услуги** — используется существующий компонент Services с унифицированным фоном (CSS-переменные).

## Примечание по `src/components/ui`

Компоненты в `src/components/ui` (shadcn-style) не используются на главной и страницах кейсов/конфиденциальности. Они исключены из проверки TypeScript, чтобы сборка проходила без установки всех опциональных зависимостей. При подключении этих компонентов нужно: вернуть папку в компиляцию, доустановить недостающие пакеты (например `react-day-picker` под версию API календаря) и при необходимости поправить типы.
